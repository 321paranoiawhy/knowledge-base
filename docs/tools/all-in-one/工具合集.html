await-to-js ​

  • await-to-js

await-to-js 可摆脱 try catch 写法, 完全的同步写法:

ts
interface ServerResponse {
  test: number;
}

const p = Promise.resolve({test: 123});

const [err, data] = await to<ServerResponse>(p);

console.log(err, data.test);
